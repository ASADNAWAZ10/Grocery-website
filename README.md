# 🛒 Grocery Website

A modern full-stack Grocery Website built using the MERN Stack. Users can browse products, add items to their cart, manage quantities, and place orders. The application also includes authentication, seller management, and cart calculation features.

## 🚀 Features

### User Features

* View all available grocery products
* Add products to cart
* Update product quantity in cart
* Remove products from cart
* User authentication and authorization
* Fetch authenticated user status
* Real-time cart item count
* Calculate total cart amount

### Seller Features

* Fetch seller authentication status
* Manage product listings
* Add new products
* Update existing products
* Remove products

## 📋 Functionalities

### Cart Management

* Add Product to Cart
* Update Item Quantity
* Remove Product from Cart
* Get Total Cart Amount
* Get Cart Item Count

### Product Management

* Fetch All Products
* View Product Details

### Authentication

* User Login & Registration
* Fetch User Auth Status
* Fetch Seller Status
